# 项目不足之处

- [ ]  用户表和好友表之间的**外键是用户id**，所以在客户端或是服务器的MyTcpSocket**存储用户id**才更为方便之后的操作，而不是存储用户名name。

> 因为如果存储的是name的话，涉及到已知用户id访问好友表时，需要先根据用户name查询到用户id，然后再进行操作，降低了效率。
**解决方案：也可以直接将name和id都进行存储。**
